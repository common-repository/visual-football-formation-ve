== Description ==
This plugin generates vertical football formations with the images of the players.

### Upgrade to Soccer Engine

Create match commentaries, standings tables, transfer market reports, and more with [Soccer Engine](https://daext.com/soccer-engine/), a complete solution for your soccer-related website.

### Define the positions of the players

Use the included drag and drop editor to define the position of the player. Alternatively, you can use one of the preloaded layouts:

* 4-4-2
* 5-3-2
* 4-4-1-1
* 4-4-2 Diamond
* 4-3-3
* 4-5-1
* 4-2-3-1
* 3-4-3
* 3-5-2

### Customize the number of players of each team

Define the number of players available in each team. In this way, you can create formations for 7 vs 7, 8 vs 8 or 11 vs 11 soccer teams based on your needs.

### Select your favorite football field or create your own

The football field is available in six different colors. You can also download the .psd version of the field <a href="https://resources.daext.com/wordpress-org/visual-football-formation-ve/field.zip">here</a>. Use this file to create football fields with custom colors or customize the field with your website logo or other additional elements.

### Customize the players

Set custom colors for the player names and the player borders based on your preferences.

### Responsive

This plugin generates responsive football formations that automatically scale based on the container.

### Suitable for High-Resolution screens

Each football field is provided with two alternative raster images of the following resolutions:

* 1084×1198
* 2168×2396

The fields are always crisp and clean with standard displays, ultra HD displays, and high-resolution mobile devices.

== Installation ==
= Installation (Single Site) =

With this procedure you will be able to install the Visual Football Formation VE plugin on your WordPress website:

1. Visit the **Plugins -> Add New** menu
2. Click on the **Upload Plugin** button and select the zip file you just downloaded
3. Click on **Install Now**
4. Click on **Activate Plugin**

= Installation (Multisite) =

This plugin supports both a **Network Activation** (the plugin will be activated on all the sites of your WordPress Network) and a **Single Site Activation** in a **WordPress Network** environment (your plugin will be activated on a single site of the network).

With this procedure you will be able to perform a **Network Activation**:

1. Visit the **Plugins -> Add New** menu
2. Click on the **Upload Plugin** button and select the zip file you just downloaded
3. Click on **Install Now**
4. Click on **Network Activate**

With this procedure you will be able to perform a **Single Site Activation** in a **WordPress Network** environment:

1. Visit the specific site of the **WordPress Network** where you want to install the plugin
2. Visit the **Plugins** menu
3. Click on the **Activate** button (just below the name of the plugin)
